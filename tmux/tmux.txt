##############
# Short Cuts #
##############

C-a c   =>  create new window
C-a n   =>  switch to next window
C-a f   =>  search for window name
C-a ,   =>  rename window
C-a o   =>  switch to sub-window
C-a :   =>  command mode

C-a [   =>  vi mode
<space> =>  mark text in vi mode
<enter> =>  copy marked text and leave vi mode
C-a ]   =>  past maked text

############
# Commands #
############

split-window -v     =>      vertical split
split-window -h     =>      horizontal split
